# Nexus Library docs

Mintlify documentation site for `nexus-library`, the AI tracing and instrumentation package.

## What this docs site covers

- Manual tracing with `NexusClient` + `Span`
- LangChain callback tracing with `PrintingHandler`
- Trace tree inspection and retrieval
- Configuration and production-hardening guidance

## Local development

```bash
npm i -g mint
mint dev
```

Check links before opening a PR:

```bash
mint broken-links
```
